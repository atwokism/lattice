# lattice

A [Trello](http://www.trello.com) sideburn

Things you should do:

  * Clone:
    `git clone https://github.com/atwokism/lattice` [Enter]
  * Install:
    `cd lattice` [Enter] then
    `npm install` [Enter] then
    `vi ./app/key` [Enter] then add your API key and save.
  * Run:
    `node lattice` [Enter]
  * Browse:
    `localhost:8080`

That's it!
