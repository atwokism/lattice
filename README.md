# lattice

A [Trello](http://www.trello.com) sideburn

Things you should do:

  * Clone lattice:
    `git clone https://github.com/atwokism/lattice` [Enter]
  * Install dependencies:
    `cd lattice` [Enter] then
    `npm install` [Enter]
  * Run:
    `node lattice` [Enter]
  * Browse:
    `localhost:8080`

That's it!
